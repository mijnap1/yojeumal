# 요즘말 — Korean Slang Timeline

2020년부터 현재까지 한국에서 실제로 사용된 유행어와 인터넷 밈을 시간순으로 살펴보는 인터랙티브 웹 아카이브입니다.

각 항목을 선택하면 뜻, 유래, 활용 상황과 예문을 확인할 수 있습니다. 날짜는 원본 콘텐츠의 공개일 또는 유행이 확인된 대표 기록일을 기준으로 정리합니다.

## Features

- 2020년부터 현재까지 이어지는 가로형 타임라인
- 유행어별 뜻, 유래, 활용 예문
- 일부 밈의 원본 장면을 카드 뒤집기로 확인하는 이미지 아카이브
- 밀집된 시기의 문구가 겹치지 않는 자동 배치
- 참고 기사와 기록을 정리한 출처 페이지
- 별도 빌드 과정이 필요 없는 HTML, CSS, JavaScript 구성

## Run locally

저장소 폴더에서 간단한 로컬 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 [http://localhost:8000](http://localhost:8000)을 엽니다.

## Project structure

```text
.
├── index.html          # 타임라인
├── script.js           # 유행어 데이터와 인터랙션
├── styles.css          # 레이아웃과 스타일
├── about.html          # 프로젝트 소개
├── sources.html        # 참고 출처
├── assets/             # 밈 이미지 아카이브
└── yojeumal-icon.png   # 사이트 아이콘
```

## Data principles

- 당시의 기사, 방송, 원본 콘텐츠 등 사용 기록을 확인할 수 있는 표현을 우선합니다.
- 최초 발생일이 불명확하면 유행이 널리 확인된 대표 날짜를 사용합니다.
- 조롱성이나 거친 표현은 의미와 사용 맥락을 함께 설명합니다.
- 출처는 [`sources.html`](sources.html)에 정리합니다.
